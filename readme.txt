# Emotion-to-Music

## Description
Cette application détecte une émotion à partir d'une image et propose des playlists Spotify en fonction de cette émotion.

## Technologies utilisées
- Flask
- DeepFace
- Neo4j
- Spotipy
- Ngrok

## Instructions
1. Clonez ce dépôt : `git clone https://github.com/votre-utilisateur/Emotion-to-Music.git`
2. Installez les dépendances : `pip install -r requirements.txt`
3. Lancez l'application : `python app.py` // il est necessaire de lancer tout le code dans Untitled37
4. Accédez à l'application via l'URL ngrok générée5 propre de vous
5. N'oubliez pas de changer les tokens , (client id , secret id ) pour api spotify
